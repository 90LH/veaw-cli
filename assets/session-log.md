# Session Log

This file stores AI development session history for VEAW.

Use `veaw session start <title>`, `veaw session log <content>`, `veaw session end [summary]`, and `veaw session list` to manage this file.

## Sessions

New sessions are appended below in a stable Markdown block format.
The example uses placeholder markers so it is not parsed as a real session:

```markdown
<!-- VEAW_SESSION_START_EXAMPLE -->
## Session: Example

- Status: active
- Started At: 2026-01-01T00:00:00.000Z
- Ended At: -
- Git Branch: main
- Git Commit: abc123

### Logs

- 2026-01-01T00:10:00.000Z

  Example progress note.

### Summary

No summary yet.
<!-- VEAW_SESSION_END_EXAMPLE -->
```
